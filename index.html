My Skills

Some of my Recent Projects

Mobile Text Extraction App with On-Device MLKit

I developed a Flutter mobile application that utilizes MLKit for on-device text extraction from live video streams, eliminating the need for external servers. The app empowers users to extract text in real-time directly on their mobile devices. By leveraging Google's MLKit, the app ensures efficient and accurate text recognition without compromising user privacy or relying on internet connectivity.

Manual Annotation Interface for BERT Model Training

I designed and developed a manual annotation interface, inspired by Label Studio, enabling efficient data labeling for training BERT models. This interface allows annotators to drag and select corresponding labels, associating them with specific contexts. The annotated data is transformed into a custom JSON format, tailored for reinforcing a BERT model.

Image processing in flutter using native c++

I engineered an advanced image processing technique to detect colored rectangle boxes within images, seamlessly extending its capabilities to real-time video streams in Flutter. Leveraging native C++ code and the power of Flutter's Foreign Function Interface (FFI), the application processes video frames in real time, identifying and tracking colored rectangles.
